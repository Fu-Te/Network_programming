# ネットワークプログラムを試すためのリポジトリ

# 注意事項
ポートスキャン等，不正アクセスにも使える技術が実装されているが，決して第3者のサーバに対して実行しないようにしてください．不正アクセスの事前調査とみなされる可能性があります．
